## 动态规划

![](https://cdn.jsdelivr.net/gh/youngjuning/images/202111232145187.png)

- 时间复杂度：<Badge>O(n)</Badge>
- 耗时：<Badge>60ms</Badge>
- 打败：<Badge>94.7%</Badge>
- 圈复杂度：<Badge>3</Badge>

```js
/**
 * @param {number} n
 * @return {number}
 */
const climbStairs = function (n) {
  let p = 0;
  let q = 0;
  let r = 1;
  for (let i = 1; i <= n; i++) {
    p = q;
    q = r;
    r = p + q;
  }
  return r;
};
```
