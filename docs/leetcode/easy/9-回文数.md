## ~~

- 时间复杂度：<Badge>O(n)</Badge>
- 耗时：<Badge>140ms</Badge>
- 圈复杂度：<Badge>5</Badge>

```js
/**
 * @param {number} x
 * @return {boolean}
 */
const isPalindrome = function (x) {
  if (x < 0) {
    return false;
  }
  let y = x;
  let z = 0;
  while (y) {
    z = z * 10 + (y % 10);
    y = ~~(y / 10);
  }
  return x === z;
};
```

- `~~`，取整，速度比 parseInt、Math.floor 和 Math.ceil 都快
  - 对于非数字，`~~` 的结果为 0
  - 对于大于 0 的数，`~~` 相当于是 `Math.floor`
  - 对于小于 0 的数，`~~` 相当于是 `Math.ceil`
